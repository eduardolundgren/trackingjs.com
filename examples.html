Using a PS Move single controller


This example shows a color tracking of a single PS Move controller using magenta color.

Using a PS Move multiple controllers


This example shows a color tracking of multiple PS Move controllers using magenta and cyan colors.

Draw Something


This example parodies the famous Draw Something game using a PSMove controller to draw. It was done with few lines of code only.

Minecraft game (tracking.js + three.js)


This example shows tracking.js being used to develop interactive games using three.js to render the 3D scene. Using mutiple controllers, magenta moves the first person player to the left and to the right, cyan moves forward.
